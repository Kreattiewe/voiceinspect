# Voiceinspect

Debugging is better when you hear the error

## Installation

Add this line to your application's Gemfile:

    gem 'voiceinspect'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install voiceinspect

## Usage

    $ Voiceinspect.say("Welcome to development mode", { voice: "Vicki" })
